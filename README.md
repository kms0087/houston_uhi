# Houston/Harris County UHI
Final project for AES 408

This project analyzes the Urban Heat Island (UHI) effect in Harris County, Texas using Landsat satellite imagery. The goal is to identify how urban areas differ in temperature compared to vegetated areas and how this pattern changes over time. This project would be useful for urban planners, enviornmental scientist, and local government. 
The analysis uses data from August 2015, 2020, and 2025 to compare spatial and temporal patterns in urban heat.

To show UHI, I calculated it as:

UHI = Pixel Temperature - Mean Temperature of Vegetated Areas

Outputs:
- NDVI maps
- Land Surface Temperature maps
- UHI maps for 2015, 2020, and 2025
- Side by side comparison of UHI over time
